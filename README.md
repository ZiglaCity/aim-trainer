## 📌 About The Project
Aim Trainer is a web-based game that challenges users to improve their aiming speed and accuracy by clicking on targets that appear at random positions. It features different difficulty levels and dynamic scoring mechanisms to keep users engaged.

## ✨ Features
- 🎯 **Three Difficulty Levels**: Easy, Medium, and Hard
- 🏆 **Scoring System**: Measures hits per second (Easy) or accuracy percentage (Medium & Hard)
- ⏳ **Timer & Target Tracking**: Keeps track of time left and targets remaining
- 🎨 **Dynamic Styling**: Game-vibe UI with animations
- 🔄 **Restart & Back Options**: Easily restart or go back to the home screen

## 🖥️ Tech Stack
- **Frontend**: React.js
- **Styling**: CSS (Modern UI & Animations)
- **State Management**: React Hooks (useState, useEffect)

## ⚡ Installation & Setup
### 1️⃣ Clone the Repository
```sh
git clone https://github.com/ZiglaCity/aim-trainer.git
cd aim-trainer
```
### 2️⃣ Install Dependencies
```sh
npm install
```
### 3️⃣ Start the Development Server
```sh
npm start
```
This will start the app at `http://localhost:3000`

## 🚀 Deployment
To deploy, use:
```sh
npm run build
```
Then, deploy to platforms like **Vercel, Netlify, or GitHub Pages**.
